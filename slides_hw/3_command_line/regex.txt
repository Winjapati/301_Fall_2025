Useful Regular Expressions Cheatsheet
----------------------------------

🔢 Digits & Numbers
- `\d` → any digit (0–9)
  Example: `\d\d` matches "42" in "Room 42".
- `\d+` → one or more digits (whole numbers)
  Example: `\d+` matches "123" in "abc123xyz".

🔤 Words & Letters
- `\w` → any “word” character (letter, digit, underscore)
  Example: `\w` matches "A" in "Alice".
- `\w+` → one or more word characters (words)
  Example: `\w+` matches "Wonderland" in "Hello Wonderland!".

⚪ Whitespace
- `\s` → any whitespace (space, tab, newline)
  Example: `Alice\sWonderland` matches across "Alice Wonderland".
- `\s+` → one or more spaces/tabs/newlines
  Example: `\s+` matches the three spaces in "Alice   in".

🎯 Anchors
- `^` → start of line
  Example: `^Alice` matches "Alice" only if it’s at the start.
- `$` → end of line
  Example: `land$` matches "Wonderland" only if it’s at the end.

🧩 Repetition
- `*` → zero or more
  Example: `ha*` matches "h", "ha", "haa", "haaa…".
- `+` → one or more
  Example: `ha+` matches "ha", "haa", but not "h".
- `?` → zero or one
  Example: `cats?` matches "cat" and "cats".

🔀 Choice & Groups
- `(dog|cat)` → either “dog” or “cat”
  Example: `I love (dog|cat)` matches "I love dog" or "I love cat".
- `(ab)+` → one or more repetitions of "ab"
  Example: matches "ab", "abab", "ababab".

📝 Character Sets
- `[aeiou]` → any one vowel
  Example: matches "a" in "cat".
- `[^aeiou]` → any character that is NOT a vowel
  Example: matches "c" in "cat".
- `[A-Z]` → any uppercase letter
  Example: matches "A" in "Alice".

✨ Special
- `.` → any single character (except newline, usually)
  Example: `c.t` matches "cat", "cot", "c t".
- `.*` → anything (zero or more of any character)
  Example: `A.*e` matches "Alice" and "A white rabbit came".
- `[[:punct:]]` (POSIX) → any punctuation
  Example: matches "!" in "Hello!".
